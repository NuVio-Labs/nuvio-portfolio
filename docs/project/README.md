# README.md

## Projekt
NuVio Labs Portfolio Website

## Ziel
Dokumentation für Strategie, Design, Content und technische Umsetzung der offiziellen NuVio Labs Portfolio Website.

## Version 1 Pflichtumfang
- Startseite
- Arbeiten / Portfolio
- Leistungen
- Über NuVio
- Kontakt
- Impressum
- Datenschutz
- Dark Mode
- Light Mode
- Deutsch, Englisch, Niederländisch

## Dokumentation
- docs/brand: Marke, Stil, Designsystem
- docs/content: Struktur und Copy für de/en/nl
- docs/project: Technik, Plan, TODO
- docs/wireframes: Seitenaufbau
- CLAUDE.md: zentrale Agent Anweisung

## Qualitätsziel
Besucher müssen schnell verstehen, was NuVio Labs macht, welche Qualität zu erwarten ist, welche Leistungen angeboten werden und wie eine Anfrage gestartet wird.
