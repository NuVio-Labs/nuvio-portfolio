# CLAUDE.md

## Mission
Arbeite an der offiziellen Portfolio Website von NuVio Labs.

NuVio Labs ist die Webdesign und Development Marke von Axel Schurer. Die Website soll hochwertige digitale Arbeit präsentieren, Vertrauen aufbauen und lokale sowie digitale Kunden zu einer Anfrage führen.

## Pflichtumfang
- Portfolio Website für NuVio Labs
- Dark Mode und Light Mode
- Mehrsprachigkeit: Deutsch, Englisch, Niederländisch
- Mobile First
- Performance, SEO und Accessibility
- DSGVO bewusste Umsetzung

## Designrichtung
Apple x NuVio: minimalistisch, hochwertig, ruhig, technisch, viel Freiraum, Black/Ivory/Champagne Gold, dezente Glassmorph Flächen.

Gold nur sparsam einsetzen für CTAs, aktive States, Icons, feine Linien und Highlights.

## Coding Regeln
- TypeScript sauber verwenden
- keine sichtbaren Texte hart in Komponenten schreiben
- i18n für de, en, nl nutzen
- Theme Farben über Tokens verwenden, keine verstreuten Hex Codes
- mobile first bauen
- Komponenten klein, sprechend und wartbar halten
- keine unnötigen Dependencies
- Animationen nur subtil mit Mehrwert

## Definition of Done
Eine Aufgabe ist fertig, wenn sie responsive funktioniert, Dark und Light Mode geprüft sind, DE/EN/NL berücksichtigt sind, keine offensichtlichen TypeScript oder Lint Fehler entstehen und die UI hochwertig, ruhig und NuVio konform wirkt.
