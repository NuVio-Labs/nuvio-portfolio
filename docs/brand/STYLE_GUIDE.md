# STYLE_GUIDE.md

## Projekt
NuVio Labs Portfolio Website

## Stilprinzip
Apple x NuVio: minimalistisch, hochwertig, technisch, ruhig, präzise, mit feinen Goldakzenten und viel Freiraum.

## Dark Mode Primärfarben
```txt
nuvio-black: #050505
graphite: #111111
charcoal: #1A1A1A
surface: #232323
gold-300: #F0D88A
gold-500: #E0B84A
gold-700: #C89A2E
champagne: #F5E7B8
text-primary: #F5F3EE
text-secondary: #B9B2A3
text-muted: #7E786C
border-soft: rgba(245,231,184,0.16)
```

## Light Mode Farben
```txt
background: #F7F4EC
surface: #FFFFFF
surface-soft: #F0EAE0
text-primary: #111111
text-secondary: #3D3932
text-muted: #746D60
accent: #C89A2E
accent-soft: #F5E7B8
border-soft: rgba(17,17,17,0.12)
```

## Farbnutzung
Gold nur für CTAs, aktive Navigation, Fokus States, Icons, feine Linien und kleine Highlights. Keine großen Goldflächen und keine bunten Primärfarben.

## Typografie
- Headings: Inter, Geist oder Manrope
- Body: Inter oder Geist Sans
- Headlines maximal semibold bis bold
- großzügige Line Height
- kurze Absätze
- klare H1 bis H3 Hierarchie

## Layout
- Mobile first
- Content Max Width: 1120px
- Wide Max Width: 1280px
- Text Max Width: 720px
- Mobile Section Padding: 72px 20px
- Desktop Section Padding: 112px 32px

## Komponenten
Buttons: Primary Gold, Secondary Border, Ghost Text.
Cards: Glassmorph im Dark Mode, klare Borders, große Radien, subtile Tiefe im Light Mode.
Header: sticky, dezent transparent, mit Theme Toggle und Sprachwechsel.
Projektkarten: Name, Branche, Ergebnisfokus, Tags, Bild oder Mockup, CTA.

## Animation
Nur subtile Fade Ups, Hover States, Border Glows und kleine Stagger Effekte. Kein Scroll Hijacking, keine hektischen Effekte.

## Accessibility
Kontraste in beiden Themes prüfen, Fokus States sichtbar halten, Alt Texte setzen, prefers-reduced-motion beachten.
