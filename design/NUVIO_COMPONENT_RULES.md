# NuVio Component Rules

## Buttons

### Primary
1. Dunkle oder gold akzentuierte Fläche
2. Klare Form
3. Radius groß aber nicht verspielt
4. Hover ruhig, leicht heller oder leicht angehoben
5. Text präzise und knapp

### Secondary
1. Transparent oder dunkle Fläche
2. Feine Border
3. Gold nur im Hover oder Focus

### Ghost
1. Sehr reduziert
2. Eher Text plus Icon
3. Nur für sekundäre Aktionen

## Cards

1. Dunkle Surface
2. Subtile Border
3. Leichte Tiefenwirkung
4. Klar definierter Innenabstand
5. Optional leichter Blur Hintergrund
6. Gold nie als Vollfläche

## Navbar

1. Schlank und hochwertig
2. Sticky wenn sinnvoll
3. Dunkler oder gläserner Hintergrund
4. Klare aktive Zustände
5. Kein visuelles Rauschen

## Hero

1. Große Typografie
2. Starker Fokus auf eine Kernbotschaft
3. Maximal zwei primäre CTAs
4. Subtile Lichteffekte
5. Keine unruhige Deko
6. Visual nur wenn es die Botschaft stärkt

## Sections

1. Deutliche Trennung durch Raum, nicht nur Linien
2. Jede Section braucht eine klare Aufgabe
3. Überschrift, Subline, Inhalt, CTA sauber staffeln
4. Keine übervollen Reihen

## Feature Grid

1. 2 bis 4 Spalten je nach Breakpoint
2. Jede Card mit klarer Aussage
3. Gute Lesbarkeit auch ohne Icon
4. Ein Highlight pro Card reicht

## Forms

1. Inputs dunkel und klar
2. Fokuszustand gold oder champagne
3. Labels lesbar und ruhig
4. Fehlermeldungen deutlich aber nicht aggressiv
5. Formulare visuell hochwertig, nicht generisch

## Footer

1. Reduziert
2. Gute Informationshierarchie
3. Dunkel, ruhig, geerdet
4. Keine unnötige Link Wand
5. Optional dezente Brand Note
