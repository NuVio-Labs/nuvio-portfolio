# NuVio Motion Rules

## Ziel von Motion

Motion soll:
1. Orientierung geben
2. Wertigkeit erhöhen
3. Fokus lenken
4. Inhalte staffeln

Motion soll nicht:
1. laut sein
2. ablenken
3. verspielt wirken
4. Performance ruinieren

## Bevorzugte Effekte

1. fade in
2. subtle y reveal
3. blur reveal leicht
4. scale minimal
5. stagger kontrolliert
6. glow shift sehr dezent

## Timing

1. Kurz bis mittel
2. Schnelle UI Reaktionen
3. Content Reveals ruhig
4. Keine extrem langen Animationen

## Hover

1. leichte Helligkeitsänderung
2. leichte Border Betonung
3. leichte Translation nach oben
4. leichter Shadow Shift

## Verbotene Effekte

1. Bounce
2. aggressive spring Physics
3. übertriebenes Rotieren
4. wilde Parallax Ketten
5. hektische Daueranimationen

## Performance Regel

Animation nie vor Nutzbarkeit.
Wenn eine Bewegung die Seite langsamer oder unruhiger macht, ist sie falsch.
