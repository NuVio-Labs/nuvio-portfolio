# NuVio UI Task Prompt

Nutze diese Vorlage, wenn du deinem Agenten eine konkrete UI Aufgabe gibst.

## Vorlage

```txt
Kontext:
Du arbeitest für die Eigenmarke NuVio. Nutze strikt das NuVio Corporate Design aus AGENTS.md und den Dateien im design Ordner.

Aufgabe:
[Beschreibe hier die konkrete UI Aufgabe]

Ziel:
[Was soll am Ende entstehen]

Technik:
[Zum Beispiel Next.js, React, Tailwind, Framer Motion]

Anforderungen:
1. Mobile First
2. Wiederverwendbare Komponenten
3. Ruhige Premium Optik
4. Black, Gold, Ivory, Glass
5. Keine bunten Primärfarben
6. Gold nur als Akzent
7. Hohe Lesbarkeit und gute Performance

Erwartete Ausgabe:
1. Direkt produktionsnaher Code
2. Kurze Erklärung der Struktur
3. Ggf. Hinweise zu Dateien und Einbau
```

## Beispiel

```txt
Kontext:
Du arbeitest für die Eigenmarke NuVio. Nutze strikt das NuVio Corporate Design aus AGENTS.md und den Dateien im design Ordner.

Aufgabe:
Erstelle eine neue Hero Section für die NuVio Startseite.

Ziel:
Die Section soll modern, ruhig, edel und conversionstark wirken.

Technik:
Next.js, React, Tailwind, Framer Motion

Anforderungen:
1. Große starke Headline
2. Subline mit klarer Positionierung
3. Zwei CTAs
4. Dunkler Hintergrund
5. Goldene Highlights nur sehr sparsam
6. Subtile Glassmorph Card oder Stats Box
7. Mobile First und responsive

Erwartete Ausgabe:
1. Vollständige React Komponente
2. Kurze Erklärung
3. Hinweise zu Klassen oder Hilfsstyles
```
